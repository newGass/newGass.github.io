◆ calculaFit()

double GA::calculaFit ( Individuo  i)

Calcula o valor de aptidão (fitness) de um indivíduo com base na distância entre seus átomos e outros critérios.

A função calculaFit() avalia a aptidão de um indivíduo na população. A aptidão é calculada com base na distância entre os átomos do cromossomo do indivíduo, comparando-as com as distâncias desejadas do template. As distâncias são calculadas usando a fórmula da distância Euclidiana.

Além disso, uma penalidade é adicionada ao valor de aptidão se dois átomos no cromossomo tiverem o mesmo ID de átomo e estiverem na mesma cadeia.

A aptidão é um valor contínuo, onde valores mais próximos de zero são considerados melhores. Se a aptidão for menor ou igual a 0.001, ela é definida como zero.

Parâmetros
indO indivíduo cuja aptidão será calculada.
Retorna
double O valor da aptidão do indivíduo. Valores mais baixos são considerados melhores, representando uma solução mais próxima do ótimo.

◆ cruza()

std::vector< Individuo > GA::cruza ( Individuo  pai1,
Individuo  pai2 
)

Realiza o cruzamento entre dois indivíduos usando o método de single-point crossover.

A função cruza() aplica a operação de cruzamento (ou "crossover") em dois indivíduos pais, gerando dois indivíduos filhos. A operação é feita com uma probabilidade especificada pela taxa de cruzamento (taxaCruzamento). Se o cruzamento ocorrer, um ponto de corte é selecionado aleatoriamente, e os genes dos pais são misturados nesse ponto para criar os filhos.

O corte é feito em um índice aleatório no cromossomo, e os genes antes desse ponto vêm de um pai, enquanto os genes após esse ponto vêm do outro pai. Isso é feito para ambos os filhos, resultando em dois cromossomos filhos que são uma mistura dos cromossomos pais.

Se o cruzamento não ocorrer (com probabilidade 1 - taxaCruzamento), os pais são retornados como os filhos sem nenhuma modificação.

Parâmetros
pai1Primeiro indivíduo para o cruzamento.
pai2Segundo indivíduo para o cruzamento.
Retorna
std::vector<Individuo> Vetor contendo os dois indivíduos filhos resultantes do cruzamento.

◆ muta()

void GA::muta ( Individuo &  ind,
int  quadrante 
)

Realiza uma mutação em um indivíduo da população, substituindo um aminoácido em uma posição selecionada aleatoriamente.

A função muta() realiza uma operação de mutação em um indivíduo específico. A mutação envolve a seleção aleatória de uma posição no cromossomo do indivíduo e a substituição do aminoácido naquela posição por outro do mesmo tipo no repositório para o quadrante especificado.

A seleção do novo aminoácido é feita com base em uma distribuição uniforme, e após a mutação, o valor de aptidão (fitness) do indivíduo é recalculado usando a função calculaFit().

A mutação é uma operação fundamental em algoritmos genéticos que ajuda a introduzir diversidade na população e permite a exploração do espaço de busca.

Parâmetros
indUma referência para o indivíduo que será mutado. O indivíduo é modificado diretamente pela função.
quadranteO quadrante no qual o novo aminoácido será buscado. É usado para determinar a disponibilidade de aminoácidos no repositório.

◆ possivel()

bool GA::possivel ( int  quad)

Verifica se é possível gerar um indivíduo válido no quadrante especificado, com base na disponibilidade de aminoácidos no repositório.

A função possivel() verifica se o repositório tem a quantidade de aminoácidos necessária para o template. Ela faz isso comparando a contagem de cada tipo de aminoácido no template (temp) com a quantidade disponível no repositório para o quadrante especificado.

Os códigos dos 20 aminoácidos padrão são verificados, e a contagem de cada um é comparada com a quantidade disponível no repositório para o quadrante dado. Se a contagem de qualquer aminoácido no template exceder a quantidade disponível no repositório para esse quadrante, a função retorna false, indicando que não é possível gerar um indivíduo válido.

Se a contagem de todos os aminoácidos no template for menor ou igual à quantidade disponível no repositório, a função retorna true, indicando que é possível gerar um indivíduo válido.

Parâmetros
quadranteO quadrante para o qual a verificação deve ser feita. Refere-se a uma região específica do repositório onde os aminoácidos estão sendo verificados.
Retorna
bool Retorna verdadeiro se for possível gerar um indivíduo válido no quadrante especificado; caso contrário, retorna falso.

◆ torneio()

Individuo GA::torneio ( )

Realiza uma seleção de torneio entre um subconjunto de indivíduos da população, retornando o indivíduo com a melhor aptidão.

A função torneio() é uma técnica de seleção utilizada em algoritmos genéticos, onde um subconjunto de indivíduos é escolhido aleatoriamente da população, e o indivíduo com a melhor aptidão é selecionado para a próxima geração.

A função primeiro seleciona aleatoriamente uma posição inicial dentro da população, garantindo que o tamanho do torneio não exceda os limites da população. A partir dessa posição inicial, são considerados os indivíduos que se encontram nas próximas tamanhoTorneio posições.

O indivíduo com a menor aptidão entre esses é selecionado como o vencedor do torneio e retornado pela função.

É importante observar que esta implementação pressupõe uma minimização da aptidão, de modo que indivíduos com aptidão menor são preferidos.

Retorna
Individuo O vencedor do torneio, ou seja, o indivíduo com a melhor (menor) aptidão dentre os considerados no torneio.

Realiza substituições conservativas nas cadeias de aminoácidos do indivíduo fornecido, de acordo com a taxa de mutação e o quadrante especificado.

A função trocaAminosEquivalentes() trabalha com o conceito de substituições conservativas, onde um aminoácido é substituído por outro com propriedades semelhantes. A função percorre cada aminoácido do cromossomo do indivíduo e, com uma probabilidade determinada pela taxa de mutação, realiza uma substituição conservativa.

A seleção do aminoácido substituto é baseada nas substituições disponíveis para cada aminoácido especificado pelo template (temp). Se houver substituições conservativas disponíveis, um aminoácido substituto é selecionado aleatoriamente da lista de substituições.

A substituição é realizada apenas se a taxa de mutação for satisfeita (ou seja, um número aleatório gerado for menor que a taxa de mutação) e se houver substituições disponíveis para o aminoácido em questão. O aminoácido substituto é então selecionado do repositório, de acordo com o quadrante especificado.

Após a realização das substituições, o cromossomo do indivíduo é atualizado com os aminoácidos substitutos, e a aptidão do indivíduo é recalculada.

Parâmetros
indReferência ao indivíduo cujas cadeias de aminoácidos serão submetidas às substituições conservativas.
quadranteO quadrante que define a região do repositório de onde os aminoácidos substitutos serão selecionados.
