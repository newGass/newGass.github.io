◆ run() [2/4]

void GASS::run ( std::vector< site > &  templates,
Repositorio &  repositorio 
)

Executa o algoritmo genético (GA) para diversos templates e quadrantes, consolidando os resultados.

A função run() orquestra a execução paralela do algoritmo genético em vários quadrantes para uma coleção de templates. Cada quadrante é processado em uma thread separada, com a execução do GA correspondente. Depois de todas as threads serem concluídas, os resultados são consolidados em uma população final que é usada para executar uma rodada adicional do GA.

Essa abordagem permite explorar várias soluções em paralelo em diferentes quadrantes e combinar os resultados, fornecendo uma análise abrangente e soluções otimizadas para os templates fornecidos.

Parâmetros
templatesVetor de objetos site, representando os templates de aminoácidos.
repositorioReferência para o repositório de aminoácidos disponíveis.

◆ runOneToOne() [1/2]

void GASS::runOneToOne ( site *  temp,
Repositorio *  repositorio,
std::set< Individuo > *  results 
)

Executa um algoritmo genético (GA) em paralelo em múltiplos quadrantes e depois consolida os resultados.

A função runOneToOne() executa uma simulação de algoritmo genético em quatro quadrantes separados utilizando múltiplas threads. Cada thread executa o algoritmo genético de acordo com os parâmetros especificados e guarda os melhores indivíduos encontrados. Depois de todas as threads serem concluídas, os resultados são consolidados, e uma nova rodada do algoritmo genético é executada com a população combinada.

Isso permite explorar várias soluções em paralelo e combinar os resultados, potencialmente levando a soluções mais robustas e diversificadas.

Parâmetros
tempPonteiro para o objeto site, representando o template de aminoácidos.
repositorioPonteiro para o repositório de aminoácidos disponíveis.
resultsPonteiro para um conjunto onde os melhores indivíduos de cada quadrante serão armazenados.
