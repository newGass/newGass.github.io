Descrição detalhada

Classe que representa um individuo.

Esta classe representa um individuo, que é uma solucao para o problema proposto. Cada individuo possui um genotipo, que é representado por um vetor de atomos, e uma aptidao, que é calculada a partir deste genotipo.
