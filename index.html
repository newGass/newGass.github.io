Algoritmo Genético para Busca de Sítios (GASS)

O projeto GASS (Genetic Algorithm Site Search) fornece uma implementação de algoritmo genético especializado na manipulação de estruturas de aminoácidos. Utiliza técnicas modernas de otimização e paralelismo para fornecer soluções precisas e eficientes para o problema.

Características

  • Paralelismo: Execução paralela em diferentes quadrantes para otimizar o tempo de processamento.
  • Manipulação de Aminoácidos: Manipula estruturas de aminoácidos, permitindo mutações, cálculo de fitness, validações, etc.
  • Flexibilidade: Parametrização para controle sobre o tamanho da população, taxa de mutação, cruzamento e outras variáveis do algoritmo genético.

Dependências

  • C++11 ou superior
  • Biblioteca STL para multithreading e containers

Classes e Estruturas

Classe GA

Responsável por gerenciar o algoritmo genético, incluindo operações como mutação, cruzamento, seleção, validação e outros processos fundamentais do algoritmo.

Classe Individuo

Representa um indivíduo dentro da população do algoritmo genético, contendo um cromossomo e métodos associados para manipulação.

Estrutura site

Representa o sítio onde o algoritmo é aplicado, contendo informações sobre os aminoácidos e suas relações.

Classe Repositorio

Armazena os aminoácidos e informações associadas, fornecendo acesso eficiente para as operações do algoritmo genético.

Namespace GASS

Contém funções auxiliares e principais para a execução do algoritmo, incluindo runOneToOne e run, que coordenam a execução em paralelo e o fluxo geral do algoritmo.

Uso

O projeto aceita uma variedade de parâmetros para ajustar o comportamento do algoritmo genético e várias funções auxiliares, sendo a interface principal o namespace GASS.

Um exemplo de uso é o seguinte:

void run(site temp, Repositorio rep, string outputFilePath){
std::set<Individuo> results;
ofstream ofs(outputFilePath);
GASS::runOneToOne(&temp, &rep, &results, getDefaultParameters());
}
